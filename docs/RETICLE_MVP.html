01 — the problem

Every screen speaks a different language.

  • 2157 public BioGRID ORCS CRISPR screens, run by hundreds of labs
  • ~58 different score types, with sign conventions that disagree
  • dir direction is often buried in free-text notes, not the data
  • mix fitness, stress, and reporter assays all pooled together
A scientist can read one screen in a few clicks — but nothing lets you ask what a gene does across all of them.
01 — the goal

Put every screen on one axis.

Turn 2,157 incomparable screens into a single calibrated coordinate system — deterministic, reproducible, auditable — so any gene returns a reliable answer across the whole corpus.

02.1 — directionality

The sign is the hard part.

A score's name doesn't encode its sign convention — so naïve pipelines, and even LLMs, get it confidently wrong.

Same input → same answer, every time
  • fixEssential-gene anchor. POLR2A, ribosome, proteasome must deplete in a viability screen.
  • →If those land on the wrong side, the screen is inverted — flip it. Deterministic & auditable.
  • llmThe LLM resolves only the genuinely ambiguous screens — then we freeze the result.
